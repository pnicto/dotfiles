# dotfiles

# Quickstart

```bash
sh -c "$(curl -fsLS get.chezmoi.io)" -- init --apply pnicto
yay -S --needed packages.txt
```

File names are different from usual because of chezmoi.

# Overview

- Terminal - alacritty and tmux (used to use kitty)
- Window manager - i3
- Bar - polybar
